# 组件规范

本文档定义 Radix UI + shadcn/ui 组件库的使用方式和最佳实践。

本文示例不包含代码注释，原示例注释已移除，如需保留请自行补充。

## 组件安装

使用 shadcn/ui CLI 添加组件：

```bash
# 添加按钮组件
pnpm dlx shadcn add button

# 添加对话框组件
pnpm dlx shadcn add dialog

# 查看所有可用组件
pnpm dlx shadcn --help
```

## 组件引用

```tsx
// shadcn/ui 组件
import { Button } from '@/components/ui/button';
import { Card, CardHeader, CardContent } from '@/components/ui/card';
import { Input } from '@/components/ui/input';
import { Badge } from '@/components/ui/badge';
import { Skeleton } from '@/components/ui/skeleton';
import {
  Dialog,
  DialogContent,
  DialogHeader,
  DialogTitle,
} from '@/components/ui/dialog';

// Lucide 图标
import { Plus, Trash2, Settings } from 'lucide-react';
```

## 常用组件

| 组件       | 用途     | 示例                                              |
| ---------- | -------- | ------------------------------------------------- |
| Button     | 按钮     | `<Button variant="default">提交</Button>`         |
| Card       | 卡片容器 | `<Card><CardHeader /><CardContent /></Card>`      |
| Input      | 输入框   | `<Input placeholder="请输入" />`                  |
| Label      | 标签     | `<Label htmlFor="name">名称</Label>`              |
| Badge      | 徽章标签 | `<Badge variant="secondary">标签</Badge>`         |
| Skeleton   | 骨架屏   | `<Skeleton className="h-4 w-full" />`             |
| Dialog     | 对话框   | `<Dialog open={open}><DialogContent /></Dialog>`  |
| DropdownMenu | 下拉菜单 | `<DropdownMenu>...</DropdownMenu>`                |
| Tabs       | 标签页   | `<Tabs><TabsList><TabsTrigger /></TabsList></Tabs>` |
| Select     | 选择器   | `<Select><SelectTrigger /><SelectContent /></Select>` |

## Button 使用场景

```tsx
import { Button } from '@/components/ui/button';
import { Plus, Loader2 } from 'lucide-react';

function ButtonExamples() {
  return (
    <div className="flex gap-2">
      <Button>默认按钮</Button>
      <Button variant="secondary">次要按钮</Button>
      <Button variant="outline">边框按钮</Button>
      <Button variant="ghost">幽灵按钮</Button>
      <Button variant="destructive">危险按钮</Button>
      <Button variant="link">链接按钮</Button>
      <Button size="sm">小按钮</Button>
      <Button size="lg">大按钮</Button>
      <Button disabled>
        <Loader2 className="mr-2 h-4 w-4 animate-spin" />
        加载中
      </Button>
      <Button>
        <Plus className="mr-2 h-4 w-4" />
        新增
      </Button>
    </div>
  );
}
```

## 表单使用示例

```tsx
import { Button } from '@/components/ui/button';
import { Input } from '@/components/ui/input';
import { Label } from '@/components/ui/label';
import { useState } from 'react';

function RepoForm() {
  const [name, setName] = useState('');
  const [url, setUrl] = useState('');

  const handleSubmit = (e: React.FormEvent) => {
    e.preventDefault();
    // 提交逻辑
  };

  return (
    <form onSubmit={handleSubmit} className="space-y-4">
      <div className="space-y-2">
        <Label htmlFor="name">仓库名称</Label>
        <Input
          id="name"
          value={name}
          onChange={(e) => setName(e.target.value)}
          placeholder="请输入仓库名称"
          required
        />
      </div>
      <div className="space-y-2">
        <Label htmlFor="url">仓库地址</Label>
        <Input
          id="url"
          type="url"
          value={url}
          onChange={(e) => setUrl(e.target.value)}
          placeholder="https://github.com/..."
          required
        />
      </div>
      <Button type="submit">提交</Button>
    </form>
  );
}
```

## Dialog 使用示例

```tsx
import { Button } from '@/components/ui/button';
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from '@/components/ui/dialog';
import { useState } from 'react';

function DeleteConfirm() {
  const [open, setOpen] = useState(false);

  return (
    <Dialog open={open} onOpenChange={setOpen}>
      <DialogTrigger asChild>
        <Button variant="destructive">删除</Button>
      </DialogTrigger>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>确认删除</DialogTitle>
          <DialogDescription>
            删除后无法恢复，请谨慎操作。
          </DialogDescription>
        </DialogHeader>
        <DialogFooter>
          <Button variant="outline" onClick={() => setOpen(false)}>
            取消
          </Button>
          <Button variant="destructive" onClick={() => setOpen(false)}>
            确认删除
          </Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```

## 自定义主题

shadcn/ui 组件基于 CSS 变量主题，在 `styles/globals.css` 中配置：

```css
@theme inline {
  --color-background: var(--background);
  --color-foreground: var(--foreground);
  --color-card: var(--card);
  --color-card-foreground: var(--card-foreground);
  --color-primary: var(--primary);
  --color-primary-foreground: var(--primary-foreground);
  /* ... */
}
```
