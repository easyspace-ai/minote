# 样式规范

本文档定义 Tailwind CSS 的样式与布局规范，强调使用原子化 CSS 与 CSS 变量。

本文示例不包含代码注释，原示例注释已移除，如需保留请自行补充。

## 响应式要求

所有页面必须使用响应式布局，确保在不同屏幕尺寸下正常显示。

Tailwind 响应式断点：
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px
- `2xl`: 1536px

## 布局与间距

使用 Tailwind 的 Flexbox 和 Grid 工具类进行布局。

```tsx
function LayoutSample() {
  return (
    <div className="flex flex-col gap-4 p-4">
      <div className="grid grid-cols-1 gap-4 md:grid-cols-2">
        <div className="rounded-lg bg-card p-4">左侧</div>
        <div className="rounded-lg bg-card p-4">右侧</div>
      </div>
    </div>
  );
}
```

## 间距规范

| 场景           | 类名示例           |
| -------------- | ------------------ |
| 页面内边距     | `p-4`, `px-6`      |
| 容器间距       | `gap-4`, `space-y-4` |
| 卡片内边距     | `p-4`, `p-6`       |
| 元素间距       | `m-2`, `my-4`      |

使用 Tailwind 的排版工具类。

## 类名命名规范

### 组合类名使用 cn()

```tsx
import { cn } from '@/core/lib/utils';

function Button({ className, variant, ...props }: ButtonProps) {
  return (
    <button
      className={cn(
        'inline-flex items-center justify-center rounded-md px-4 py-2',
        variant === 'primary' && 'bg-primary text-primary-foreground',
        variant === 'secondary' && 'bg-secondary text-secondary-foreground',
        className
      )}
      {...props}
    />
  );
}
```

### 类名排序规范

按以下优先级排序：
1. 布局（display, position, flex, grid）
2. 尺寸（width, height, padding, margin）
3. 外观（background, border, shadow）
4. 排版（font, text, leading）
5. 交互（cursor, hover, focus）
6. 响应式（sm:, md:, lg:）
7. 状态（disabled:, aria-）

```tsx
<button className="flex h-10 items-center justify-center rounded-md bg-primary px-4 text-sm font-medium text-primary-foreground hover:bg-primary/90 disabled:pointer-events-none disabled:opacity-50">
  按钮
</button>
```

## 表单与按钮样式

使用 shadcn/ui 组件和 Tailwind 工具类。

```tsx
import { Button } from '@/components/ui/button';
import { Input } from '@/components/ui/input';
import { Label } from '@/components/ui/label';

function FormSample() {
  return (
    <form className="space-y-4">
      <div className="space-y-2">
        <Label htmlFor="name">名称</Label>
        <Input id="name" placeholder="请输入名称" />
      </div>
      <Button type="submit" className="w-full">
        提交
      </Button>
    </form>
  );
}
```

## 自定义样式范围

如需定制，建议使用组件的 `className` 属性，范围限定到局部区域，不覆盖全局样式。

避免：
```css
/* 不要这样做 */
.ant-btn { background: red; }
```

推荐：
```tsx
<Button className="bg-red-500 hover:bg-red-600">自定义按钮</Button>
```
