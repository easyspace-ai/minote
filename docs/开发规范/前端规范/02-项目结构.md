# 项目结构

本文档定义 YouMind 前端项目的目录组织和文件布局。

## 目录结构

```
web/
├── src/
│   ├── components/              # 组件目录
│   │   ├── ui/                  # shadcn/ui 组件（自动生成）
│   │   ├── workspace/           # 工作空间相关组件
│   │   ├── layout/              # 布局组件
│   │   └── common/              # 业务通用组件
│   ├── pages/                   # 页面组件
│   ├── core/                    # 核心模块
│   │   ├── api/                 # API 客户端和请求
│   │   ├── hooks/               # 全局共享 Hooks
│   │   ├── lib/                 # 工具函数
│   │   └── types/               # 全局类型定义
│   ├── i18n/                    # 国际化配置
│   │   ├── locales/             # 语言包
│   │   │   ├── zh-CN.json       # 简体中文
│   │   │   └── en-US.json       # 英文
│   │   └── index.ts             # i18n 初始化
│   ├── providers/               # Context Providers
│   │   ├── ThemeProvider.tsx
│   │   └── QueryProvider.tsx
│   ├── styles/                  # 全局样式
│   │   └── globals.css          # Tailwind 入口
│   ├── App.tsx                  # 根组件
│   └── main.tsx                 # 入口文件
├── components.json              # shadcn/ui 配置
├── tailwind.config.ts           # Tailwind 配置
├── vite.config.ts               # Vite 配置
└── package.json
```

## 目录说明

### components/

- `ui/`：由 shadcn/ui CLI 自动生成，不要手动修改。使用 `pnpm dlx shadcn add <component>` 添加组件
- `workspace/`：工作空间相关的业务组件
- `layout/`：布局相关组件，如 Header、Sidebar、Footer
- `common/`：业务通用组件，如 LanguageSwitcher、ThemeSwitcher

### pages/

页面级组件，与路由一一对应。每个页面组件使用 `export default`。

```tsx
// pages/Dashboard.tsx
export default function Dashboard() {
  return <div>Dashboard Page</div>;
}
```

### core/

核心业务逻辑模块，替代传统的 services/hooks/types 分散目录。

- `api/`：TanStack Query 的 query/mutation 定义、API 客户端
- `hooks/`：全局共享的自定义 Hooks
- `lib/`：通用工具函数，如 `cn()` 类名合并函数
- `types/`：全局 TypeScript 类型定义

### i18n/

国际化配置和语言包文件，使用 React i18n 标准格式。

### providers/

React Context Providers，如主题、查询客户端等。

### styles/

全局样式和 CSS 变量定义。Tailwind CSS 的入口文件在此。

## 文件命名规范

- 组件文件：PascalCase（如 `Button.tsx`, `WorkspaceCard.tsx`）
- 工具文件：camelCase（如 `utils.ts`, `useAuth.ts`）
- 类型文件：kebab-case.types.ts（如 `api.types.ts`）
- 常量文件：SCREAMING_SNAKE_CASE 或 camelCase（如 `CONSTANTS.ts` 或 `constants.ts`）
