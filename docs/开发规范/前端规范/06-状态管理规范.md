# 状态管理规范

本文档定义 React 状态管理的分类、原则和模式，以 TanStack Query 为核心。

## 状态分类

| 类型         | 方案                    | 示例                   |
| ------------ | ----------------------- | ---------------------- |
| 组件状态     | useState / useReducer   | 表单输入、开关状态     |
| 服务端状态   | TanStack Query          | API 数据、缓存         |
| 跨组件状态   | Context + useReducer    | 用户信息、主题设置     |
| URL 状态     | useSearchParams         | 筛选条件、分页         |

## 状态提升原则

- 状态应放在最近的共同父组件
- 避免过度使用全局状态
- 优先使用组件组合而非 Context
- 服务端状态优先使用 TanStack Query

## TanStack Query 数据获取

### Query 定义

```tsx
// core/api/queries.ts
import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';

// 获取仓库列表
export function useRepositories() {
  return useQuery({
    queryKey: ['repositories'],
    queryFn: async () => {
      const response = await fetch('/api/repositories');
      if (!response.ok) throw new Error('Failed to fetch');
      return response.json();
    },
  });
}

// 创建仓库
export function useCreateRepository() {
  const queryClient = useQueryClient();
  
  return useMutation({
    mutationFn: async (data: CreateRepoInput) => {
      const response = await fetch('/api/repositories', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(data),
      });
      if (!response.ok) throw new Error('Failed to create');
      return response.json();
    },
    onSuccess: () => {
      // 成功后刷新仓库列表
      queryClient.invalidateQueries({ queryKey: ['repositories'] });
    },
  });
}
```

### 使用示例

```tsx
import { useRepositories, useCreateRepository } from '@/core/api/queries';
import { Skeleton } from '@/components/ui/skeleton';
import { Button } from '@/components/ui/button';

function RepositoryList() {
  const { data: repos, isLoading, error } = useRepositories();
  const createRepo = useCreateRepository();

  if (isLoading) return <Skeleton className="h-32" />;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <ul>
        {repos?.map((repo) => (
          <li key={repo.id}>{repo.name}</li>
        ))}
      </ul>
      <Button 
        onClick={() => createRepo.mutate({ name: 'new-repo' })}
        disabled={createRepo.isPending}
      >
        {createRepo.isPending ? '创建中...' : '创建仓库'}
      </Button>
    </div>
  );
}
```

### Query Key 规范

```ts
// 按资源层级组织 queryKey
['repositories']                          // 所有仓库
['repositories', id]                      // 单个仓库
['repositories', id, 'tasks']             // 仓库的任务
['repositories', id, 'tasks', taskId]     // 仓库的单个任务
['workspaces']                            // 工作空间
['workspaces', id]                        // 单个工作空间
```

### 高级用法

```tsx
// 分页查询
function usePaginatedRepositories(page: number) {
  return useQuery({
    queryKey: ['repositories', 'page', page],
    queryFn: () => fetchRepositories({ page }),
    placeholderData: (previousData) => previousData, // 保持旧数据
  });
}

// 轮询
function useTaskStatus(taskId: string) {
  return useQuery({
    queryKey: ['tasks', taskId, 'status'],
    queryFn: () => fetchTaskStatus(taskId),
    refetchInterval: 5000, // 每 5 秒刷新
  });
}

// 乐观更新
function useUpdateTask() {
  const queryClient = useQueryClient();
  
  return useMutation({
    mutationFn: updateTask,
    onMutate: async (newTask) => {
      await queryClient.cancelQueries({ queryKey: ['tasks', newTask.id] });
      const previousTask = queryClient.getQueryData(['tasks', newTask.id]);
      queryClient.setQueryData(['tasks', newTask.id], newTask);
      return { previousTask };
    },
    onError: (err, newTask, context) => {
      queryClient.setQueryData(['tasks', newTask.id], context?.previousTask);
    },
    onSettled: (newTask) => {
      queryClient.invalidateQueries({ queryKey: ['tasks', newTask?.id] });
    },
  });
}
```

## Context 使用场景

仅在以下场景使用 Context：

1. **主题切换**：全局主题状态（配合 next-themes）
2. **国际化**：语言设置
3. **用户认证**：登录状态（如需要全局访问）

## Context 示例

```tsx
// contexts/AppContext.tsx
import { createContext, useContext, useState, ReactNode } from 'react';

interface AppState {
  sidebarOpen: boolean;
}

interface AppContextType {
  state: AppState;
  toggleSidebar: () => void;
}

const AppContext = createContext<AppContextType | null>(null);

export function AppProvider({ children }: { children: ReactNode }) {
  const [state, setState] = useState<AppState>({
    sidebarOpen: true,
  });

  const toggleSidebar = () => {
    setState((prev) => ({ ...prev, sidebarOpen: !prev.sidebarOpen }));
  };

  return (
    <AppContext.Provider value={{ state, toggleSidebar }}>
      {children}
    </AppContext.Provider>
  );
}

export function useApp() {
  const context = useContext(AppContext);
  if (!context) {
    throw new Error('useApp must be used within AppProvider');
  }
  return context;
}
```

## 禁用 Redux / Zustand

本项目不使用 Redux 或 Zustand 进行状态管理。所有服务端状态都应使用 TanStack Query。

如需客户端全局状态，优先选择：
1. URL 参数（筛选、分页）
2. Context（主题、认证）
3. 组件状态提升
