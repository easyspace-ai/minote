# 技术栈

本文档定义 YouMind 前端项目使用的技术栈和版本要求。

## 包管理

本项目使用 pnpm 作为包管理工具。
一切依赖都通过 pnpm 安装和管理。
一切命令都通过 pnpm 执行。

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 核心技术

| 技术           | 版本    | 用途           |
| -------------- | ------- | -------------- |
| React          | 19.x    | UI 框架        |
| TypeScript     | 5.8.x   | 类型安全       |
| Vite           | 6.x     | 构建工具       |
| Tailwind CSS   | 4.x     | 原子化 CSS     |
| Radix UI       | 1.x     | 无障碍 UI 原语 |
| shadcn/ui      | 2.x     | UI 组件库      |
| React Router   | 7.x     | 路由管理       |
| TanStack Query | 5.x     | 服务端状态管理 |
| Lucide React   | 0.x     | 图标库         |

## 开发环境要求

| 工具       | 版本        | 说明           |
| ---------- | ----------- | -------------- |
| Node.js    | >= 20.x LTS | 运行环境       |
| pnpm       | >= 9.x      | 包管理器       |

## 技术选型说明

### React 19
- 使用最新的 React 版本，包含并发特性和自动批处理优化
- 支持 Server Components（未来扩展）

### Tailwind CSS 4
- 新一代 Tailwind，性能大幅提升
- 基于 CSS 原生 @layer 和 @property
- 无需配置文件，基于 CSS 的 theme() 函数

### Radix UI + shadcn/ui
- Radix 提供无障碍、可访问的 UI 原语
- shadcn/ui 基于 Radix 构建精美的预制组件
- 组件代码直接集成到项目中，完全可定制
- 非 npm 包形式，避免版本锁定问题

### TanStack Query
- 专门用于服务端状态管理
- 自动缓存、重试、轮询、乐观更新
- 替代 Redux/Zustand 管理服务端数据

### React Router 7
- 最新版本，支持数据加载器和表单操作
- 可选的 SSR/SPA 模式

## 版本选择原则

1. **稳定性优先**：使用 LTS 或稳定版本
2. **兼容性考虑**：确保依赖间版本兼容
3. **安全性更新**：及时更新安全补丁
4. **最新特性**：积极采用能带来明显收益的新版本
