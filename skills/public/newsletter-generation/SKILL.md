# Newsletter Generation Skill

## Overview

This skill creates professional newsletters with structured content, engaging copy, and appropriate formatting. Use it when the user needs to produce email newsletters, company updates, or content digests.

## When to Use This Skill

- User asks to create or write a newsletter
- User needs a weekly/monthly digest
- User wants to compile news or updates into a newsletter format
- User requests an email campaign draft
- User needs a company or product update newsletter

## Newsletter Structure

### 1. Header Section
- Newsletter title/brand name
- Issue number and date
- Brief tagline or subtitle

### 2. Hero/Feature Story
- Main headline with compelling hook
- Summary paragraph (2-3 sentences)
- Call-to-action link
- Optional hero image description

### 3. Content Sections
- **News & Updates**: Latest developments
- **Featured Article**: In-depth piece
- **Tips & How-tos**: Practical advice
- **Industry Trends**: Market insights
- **Product Updates**: New features/releases
- **Community Spotlight**: User stories

### 4. Supporting Content
- Quick links / resource roundup
- Upcoming events or dates
- Social media highlights
- Statistics or data points

### 5. Footer
- Unsubscribe link reminder
- Contact information
- Social media links
- Legal disclaimers

## Writing Guidelines

### Tone & Style
- Professional yet approachable
- Scannable with clear headings
- Action-oriented language
- Consistent brand voice

### Best Practices
- **Subject line**: Craft compelling, concise subject lines (40-60 chars)
- **Preview text**: Write engaging preview text
- **Length**: Keep total content under 500 words for engagement
- **Links**: Include 3-5 key call-to-action links
- **Personalization**: Use merge tags where appropriate
- **Mobile-friendly**: Design for mobile-first reading

### Content Sourcing
- Research recent news in the relevant domain
- Curate from multiple credible sources
- Include original insights and analysis
- Credit sources appropriately

## Output Formats

- HTML email template (default)
- Markdown for content management
- Plain text fallback version

## Notes
- Consider audience segmentation
- A/B test subject lines when possible
- Include alt text for images
- Maintain consistent sending schedule
- Follow email marketing regulations (CAN-SPAM, GDPR)
